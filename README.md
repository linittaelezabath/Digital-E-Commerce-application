## Digital E-Commerce Application
Building a Full Stack Digital E-Commerce App

This project is a modern full-stack Digital E-Commerce application built using Next.js 14 and React.js for the frontend, Strapi as the backend CMS, and Tailwind CSS for responsive and clean UI design. It includes essential e-commerce features such as product listings, user authentication, dynamic content management, and an integrated email resending system to keep users informed.

## Tech Stack

Next.js 14

React.js

Strapi (Headless CMS)

Tailwind CSS

Node.js

Email Resending Integration

## Getting Started

This is a Next.js
 project bootstrapped with create-next-app
.

First, run the development server:

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev


Open http://localhost:3000
 with your browser to see the result.

You can start editing the page by modifying app/page.js. The page auto-updates as you edit the file.
